# CodeCraft
CodeCraft 💻

A Simple Coding Learning Platform

CodeCraft is a web-based platform designed to help beginners learn the fundamentals of programming in a simple and interactive way.

The project focuses on providing an easy environment where users can explore coding concepts, understand basic programming ideas, and practice learning development skills.

This project is created mainly for educational and portfolio purposes.

---

✨ Features

- 📚 Beginner-friendly coding lessons
- 💻 Clean learning interface
- ⚡ Simple and lightweight design
- 📱 Mobile responsive layout
- 🎨 Modern UI for better learning experience

---

- HTML5
- CSS3
- JavaScript
